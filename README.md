# Front-End Developer Test

## Description

 SPA for search user and repos on GitHub 

## Bootstrap

```
git clone git@github.com:JackiFlower/abr.git && cd abbr-dev-test  
git checkout master  
npm install  
```

## Start & Development 

Launch: `npm run server`    
Link: `http://localhost:8000/`

## Distribution

Launch: `npm run dist`
